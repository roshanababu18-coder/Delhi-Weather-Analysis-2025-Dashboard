# Weather Analysis Dashboard – Delhi (2025)

## Project Overview
This project presents a Weather Analysis Dashboard for Delhi (2025).
The dashboard visualizes important environmental indicators such as
temperature, air quality index (AQI), humidity, and wind speed.

The goal of this project is to help users understand environmental
trends and pollution patterns using interactive visualizations.

## Dashboard Features

### 1. Average Weather Indicators
Displays key weather statistics:
- Average Temperature (°C)
- Average AQI
- Average Wind Speed
- Average Humidity

### 2. AQI Trend Analysis
A line chart showing how AQI levels change throughout the year.

### 3. Temperature Trend
Shows increases and decreases in temperature over time.

### 4. Average Pollution Levels
Bar chart comparing pollution indicators:
- PM2.5
- PM10
- NO2

### 5. Weather Condition Distribution
Pie chart displaying the distribution of:
- Clear Sky
- Mainly Clear
- Drizzle
- Fog
- Rain

### 6. Location Filter
Users can analyze weather data for multiple locations:
- Anand Vihar
- Connaught Place
- Dwarka
- IGI Airport
- Okhla Phase III
- Rohini

### 7. Date Filter
Allows users to select custom date ranges.
